# 📰 Urdu Fake News Detector

An AI-powered web app that detects fake news in Urdu language using NLP.

## Features
- Paste any Urdu news headline
- Get instant Real/Fake classification
- Shows confidence percentage
- Built with Python and AI

## Tech Stack
Python | Streamlit | HuggingFace Transformers | NLP

## How to Run
pip install -r requirements.txt
streamlit run app.py
